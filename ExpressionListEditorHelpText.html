Expression List Editor Help

The Expression List Editor is used to manipulate multiple expressions together. An expression list can contain a maximum of 15 expressions.

Expressions, and subexpressions, may be selected using the approach demonstrated in the tutorial. Selected expressions will appear in blue in Expression List Editor while selected expressions will appear in red in the Expression Editor.

The Expression List Toobar

At the top-center of the Expression List Editor is the set of tools used to manipulate the current list of expressions. From left to right they are:
  1. Choose Expression (page button) - Choose an expression from a list or select a new list of expressions.
  2. Enter Expression ("+" button) - Add a new expression using the Expression Input Screen.
  3. Edit Expression (pencil button) - Edit the selected expression.
  4. Save Expression List (disk button) - Save the list of expressions.
  5. Delete Expression (delete button) - Delete the selected expression or all expression in the list.
These tools are described in detail below.

Choose Expression

To use Choose Expression, touch the Choose Expression button to see the table of saved expression lists. Navigate to the desired expression and select it. The chosen expression will be added to the current expression list.

Expression lists you have saved will appear in the My Expression Lists table entry.

The expression lists in My Expression Lists table entry may be edited.

Touch the "All" table entry in an expression list to load all expressions in that list into the Expression List Editor. Note that your current list of expressions will be replaced, so make sure you save your list, if desired, before using this option.

Enter Expression

To use Enter Expression, touch the Enter Expression button, enter your expression in the Expression Input Screen and touch the Done button. The entered expression will be added to the current expression list.

Refer to the help text on the Expression Editor screen for help with using the expression editor. You access the help text by pressing the (i) button in the upper righthand corner of the Expression Editor screen.

Edit Expression

If an expression, or subexpression, in your list is selected the Edit Expression button will be active. Touch the Edit Expression button to load the selected expression into the Expression Editor. When you are finished editing, the selected expression will be replaced with the new one.

Save Expression List

Touch the Save Expression List button to save the current list of expressions. Enter a name for your expression list and press the "Done" button in the keyboard. Your expression list will appear in the My Expression Lists table entry of the Choose Expression tool.

Delete Expression

If an expression, or subexpression, in your list is selected, touch the Delete Expression button to delete the expression. You will be prompted to confirm the deletion. To delete all expressions in the list, touch the Delete Expression button with no expression selected and confirm the deletion.

Substitutions

If your expression list has one or more equations, you can substitute the left or right side of the equation into any subexpression in your expression list that is symbolically equivalent. For example, if your expression list contains:
  1. y = 4*x - 3
  2. x = 3 - c

You can replace the x in the first equation with 3 - c from the second. To do so, select the x in the second equation (the left side of the equation) and then the x in the first. The x in the first equation will be replaced with 3 - c (y = 4*(3 - c) - 3).

Here are a couple of ideas for using substitution.

1) Solve the following system of equations:
  1. Solve for x and y
  2. y = 4*x - 3
  3. y = 3 - x
Select the y in the second equation and then the y in the first equation. The first equation will become 3 - x = 4*x - 3. Solve for x. Your equation list is now:
  1. x = 6/5
  2. y = 3 - x
Select the x in the first equation and then the x in the second equation. The second equation will become y = 3 - 6/5. Now solve for y. Your equation list is now:
  1. x = 6/5
  2. y = 9/5
2) Evaluate a function for a given value of x:
  1. y = 4*x - 3
  2. x = 2
Select the x in the second equation and then the x in the first equation. The first equation will become y = 4*2 - 3. Solve for y (y = 5), which is your answer.
