Pocket Algebra Help

Here's a quick overview of Pocket Algebra.

Enter Expression

Enter Expression provides a way to quickly enter a single expression for evaluation. Use this feature when you don't need to save the expression or evaluate it in conjunction with other expressions.

To use Enter expression, touch the Enter Expression button, enter your expression in the Expression Input Screen and touch the Done button. Your expression will be loaded into the Expression Editor for evaluation.

Refer to the help text on the Expression Editor screen for help with using the expression editor. You access the help text by pressing the (i) button in the upper righthand corner of the Expression Editor screen.

Touch the back (Pocket Algebra) button to return to the main screen. Note that your work will not be saved.

Choose Expression

Choose Expression provides a way to pick a saved expression for evaluation. Use this feature to select one of your saved expressions or one of the expressions provided with Pocket Algebra.

To use Choose Expression, touch the Choose Expression button to see the table of saved expression lists. Navigate to the desired expression and select it. Expression lists you have saved will appear in the My Expression Lists table entry. The chosen expression will be loaded into the Expression Editor for evaluation.

You will not have the option to select all expressions in a list when using the Choose Expression option from the main screen. Refer to the Expression List help for more information on selecting all expressions in a list.

The expression lists in My Expression Lists table entry may be edited.

Refer to the help text on the Expression Editor screen for help with using the expression editor. You access the help text by pressing the (i) button in the upper righthand corner of the Expression Editor screen.

Touch the back (Pocket Algebra) button to return to the main screen. Note that your work will not be saved.

Expression List

Expression List provides a way to evaluate multiple expressions in context of one another. Refer to the help text on the Expression List screen for further details. You access the help text by pressing the (i) button in the upper righthand corner of the Expression List screen.

Tutorial

Use the Tutorial to learn how to manipulate your expressions. Touch the Tutorial button to see the Tutorial Chapters table and choose a chapter. Use the left and right arrows to move from step to step within a tutorial chapter. Note that the tutorial will continue with the next chapter when you reach the end of the previous chapter.

Touch the back (Pocket Algebra) button on the Tutorial Chapters screen to return to the main screen.

When in the tutorial touch the Touch the back (Tutorial Chapters) button to return to the Tutorial Chapters screen.

Plan to return to the tutorial often as you become familiar with Pocket Algebra. There is a lot to learn and you probably won't use all of the features when you first start.

A Note to Students

Pocket Algebra was created to help you learn algebra. It will not teach you algebra - it assumes you are in a class or using an algebra textbook to learn. But, if properly used, it can take some of the pain out of the learning experience (assuming you're not already acing your tests, in which case Pocket Algebra will be more for fun!).

Warning! It's possible to use Pocket Algebra to solve reasonably complex equations and problems and learn nothing. Here is are a couple of suggestions for using Pocket Algebra to improve you test scores:

  • Check you work. Solve a homework problem yourself and then check your work by solving the same problem in Pocket Algebra. If you were wrong you should be able to see where you went wrong. Make of mental note of the mistake and try the problem again (without Pocket Algebra) until you get it right.

  • Repetition. Repetition. Repetition. Create lists of homework problems and repeatedly solve them using Pocket Algebra until they are second nature to you. Hopefully, Pocket Algebra can help make this process easier and a bit more fun.

If you can use Pocket Algebra effectively to solve algebra problems it should be a short step to being able to do so without it. Make sure you take that last step!
